Danysz Flutter Automation Toolkit

A powerful, zero-touch Python orchestrator designed to instantly generate a production-ready Flutter mono-repo. This toolkit eliminates hours of manual boilerplate by fully automating Google Cloud and Firebase provisioning, Android keystore generation, CI/CD deployment pipelines, and localization workflows.

Key Features:

  • ⚡ Zero-Touch Setup: Initializes your workspace, provisions cloud databases, configures FlutterFire, and pushes your initial Git commit in a single execution.
  • 🔐 Security First: Automatically generates, configures, and safely hides your 2048-bit RSA production keystores.
  • 🚀 Automated CI/CD: Injects pre-configured Bash scripts to handle semantic version bumping and automated distribution to iOS and Android testing environments.
  • 🌍 Smart Localization: Custom Python utilities that hunt down hardcoded Dart strings, move them to translation files, and automatically fix the resulting analyzer errors.
